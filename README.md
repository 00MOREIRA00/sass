## Executando scripts

Executando o Eslint

```bash
pnpm run lint
```

Executando o Prettier

```bash
pnpm run format
```

Instalando no packge.json correto

Rodando a api

```bash
pnpm dev
```

Migrando banco de dados 

```bash
pnpm prisma migrate dev
```

Instalando seed banco de dados

```bash
pnpm prisma db seed
```

Abrindo prisma studio

```
pnpm prisma studio
```
